# Predictive-UX-Friction-Analysis-Objective
Internship Assignment: Predictive UX Friction Analysis
Objective:
To evolve the current Intelligent AI Assistant project by developing a systematic method for UX assessment that identifies user frustration through behavioral data.  
Task Description:
You are tasked with designing a "Friction Mapping" component for your assistant. Instead of relying on manual feedback, this tool should use interaction signals to predict where a user is struggling.  
Key Requirements:
• Metric Definition: Develop a formula for a Friction Index (\bm{FI}) that factors in variables like click velocity, repeat attempts, and dwell time.  
• Integration: Update the app2.py logic to log these interaction events without compromising the assistant’s performance.  
• Visualization: Propose a method to highlight "hotspots"—UI areas where the AI predicts high user confusion.  
• Reporting: Create a mock diagnostic report that categorizes sessions as "Seamless," "Mild Friction," or "High Frustration".  
Deliverable:
Please provide a 2-page technical proposal and a proof-of-concept script by the end of the week.
